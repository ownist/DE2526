### অ্যাসাইনমেন্ট নোটিশ

সকল শিক্ষার্থীর অবগতির জন্য জানানো যাচ্ছে যে, Python Programming বিষয়ের উপর একটি অ্যাসাইনমেন্ট দেওয়া হয়েছে। অ্যাসাইনমেন্টে নিচের প্রোগ্রামগুলোর Python code লিখতে হবে এবং প্রতিটি প্রোগ্রামের Flowchart অঙ্কন করতে হবে।

**অ্যাসাইনমেন্টের টপিকসমূহ:**

- ১. সবচেয়ে বড় সংখ্যা নির্ণয়
- ২. সবচেয়ে ছোট সংখ্যা নির্ণয়
- ৩. জোড় বা বিজোড় সংখ্যা নির্ণয়
- ৪. ত্রিভুজের ক্ষেত্রফল নির্ণয়
- ৫. গ্রেড নির্ণয়
- ৬. অধিবর্ষ নির্ণয়
- ৭. দ্বিঘাত সমীকরণের সমাধান নির্ণয়

প্রতিটি প্রোগ্রাম পরিষ্কারভাবে খাতায় লিখতে হবে এবং Flowchart যথাযথ নিয়ম অনুসরণ করে সুন্দরভাবে অঙ্কন করতে হবে।

**জমাদানের তারিখ:** ১২ মে ২০২৬, মঙ্গলবার

**বিষয়:** Python Programming

**কাজ:** Python Code Writing and Flowchart Drawing

নির্ধারিত তারিখের মধ্যে অবশ্যই অ্যাসাইনমেন্ট জমা দিতে হবে। নির্ধারিত দিনের পর আর কোনো অ্যাসাইনমেন্ট গ্রহণ করা হবে না।
